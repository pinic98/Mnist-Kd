# Knowledge-Distillation (Pytorch)
This project is a implementation of Knowledge distillation on Mnist dataset.
  * Framework : PyTorch
  * Dataset : Mnist

## Result : "Teacher Net" to "Student Net" distillation

Model | Test Accuracy
---|---|
